I'm certain this is how he plays it live with the New Japan Philharmonic, minus any wailing on the high E's 19th fret instead of the proper arpeggio.
All notes save for the A before the first repeat bar and the last bended note are sixteenth note triplets.

This tab exists because people keep tabbing things out including mistakes. Don't do that.
Example is the B-flat-maj arpeggio at the beginning of the third bar, this needs to start with a D.

A guitar pro file will be in this directory. Suggest using TuxGuitar instead of Guitar Pro.
The spacing of the notes will be better for the tab and you'll have the music notation in correct time (not so sure about the time).
